코드를 좀 더 구분하기 쉽게 하기 위해서 스크립트 파일을 전체적으로 분할하였으며 스크립트 파일의 이름을 통해서 어떤 코드가 들어있는지 알기 쉽게 되어있습니다. 또한 캔버스에서 1,2,3번 키를 통해서 변경되는 모델은 3.model.x.name.js와 같은 이름으로 명명되어 있습니다.

  • 1번키 → 3.model.1.fallball.js
  • 2번키 → 3.model.2.moveball.js
  • 3번키 → 3.model.3.fallballs.js

3.model.2.moveball.bak.js의 경우 1차 시도를 하였을때 사용된 코드입니다. 여기서 1차 시도와 2차 시도란, '1차 시도'는 과제를 처음 시작하면서 만들었던 코드, '2차 시도'는 교수님이 주신 코드를 수정하는 것으로 진행한 것입니다.
